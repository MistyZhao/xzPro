1.项目搭建
	服务器文件，静态目录，路由器目录，数据库模块等
	初始化文件夹为模块目录（才能在当前目录下安装第三年模块）npm init -y
	切换目录，npm install 包名
		服务器文件中
			导入模块，创建服务器，开启服务器，静态目录托管，body-parser使用，路由器挂载
		数据库模块
			导入模块，创建连接池对象，指定模块的导出对象
		路由器模块
			导入模块，创建路由器，配置路由，并且实现数据库操作
			注意：路由器模块内如何定义路由，服务器挂载路由器并为路由器下所有路由地址添加前缀后，完整的访问地址如何表示
	注意：项目运行前，一定要开启数据库服务器
2.实现用户模块下的接口
	1）注册接口
		实现用户名唯一性的检测（如果被占用则无法注册）
	2）登录接口
		创建首页文件
		登陆成功之后跳转至首页
	3）检索接口
		创建前端页面，表单接受用户输入uid
		后端检测是否提供uid，并响应结果
	4) 分页查询设置默认值
